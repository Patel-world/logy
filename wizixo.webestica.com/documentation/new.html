Introduction To CSS Grid

CSS Grid is an indispensable concept that every excellent web developer should master!

While it might often look and feel more complicated to use compared to other layout systems like Flexbox, it's actually very straight-forward and simple to use once you get a hold of the syntax behind it and some of the core functionalities it offers!

Using grid can save you from creating cumbersome flex layouts with lots of containers, and also from using media queries in many cases! Besides that, you also get to feel like a CSS ninja when your grid does what you want it to do!

What Is CSS Grid

Simply put, CSS Grid is a two dimensional layout system for web pages, allowing the display of content in columns and rows. Grids allow us to create complex layouts in a fairly easy way, and it is in many cases a better choice over other methods, such as flexbox. Each grid layout will contain any number of columns and rows, and the gaps that will separate them, which are also known as gutters. Let's learn how to use it!

Basic Usage Of Grid

Let's take a look at a very simple use case for grid! 


    
      <div class="grid-container">
        <div class="grid-item"> 1 </div>
        <div class="grid-item"> 2 </div>
        <div class="grid-item"> 3 </div>
        <div class="grid-item"> 4 </div>
      </div>
    
    
  

  
     .grid-container {
       display: grid;
       grid-template-columns: 100px 100px 100px;
       gap:  1em;
        
    }

It's that simple to create a three column layout with automatic rows and an equal gap between them!

What We Did

  1. Use display: grid on the parent (container) element
  2. Use the grid-template-columns property in order to specify that our grid will contain three columns of fixed width
  3. Use the gap property to add an equal lem gap between both the rows and the columns of the grid

Note:Try to resize the colorful boxes and container(drag your cursor at the bottom end of whatever you want to resize and drag) and observe the behaviour of grid,, this is the best way understand what happend when content overflow or underflow.

Using The Fraction (fr) Unit

Instead of using fixed widths for our rows/ columns, we can make our layout more flexible by using fractions. The fr unit represents one fraction of the available space in the grid container. Using the same HTML code, we can generate two new layouts by using the fraction unit:


  
           .grid-container {
             display: grid;
             grid-template-columns: 1fr 1fr 1fr;
             gap:  1em;
              
          }

1. Layout of two rows and columns of equal size


  
             .grid-container {
               display: grid;
               grid-template-columns: 1fr 1fr;
               grid-template-rows: 1fr 1fr;
               gap:  1em;
                
            }

2. Layout of two rows and columns of different size


    
               .grid-container {
                 display: grid;
                 grid-template-columns: 1fr 2fr;
                 grid-template-rows: 2fr 1fr;
               gap:  1em;
                  
              }

As you can see, larger fractions occupy more space than the smaller fractions in the available space.

3. The repeat() function

You can use the CSS repeat() function as an alternative for specifying the number of columns you want your grid to repeat creating.


    
                 .grid-container {
                   display: grid;
                   grid-template-columns: repeat(3, 1fr);
                   grid-template-rows: 100px;
               gap:  1em;
                    
                }

Now the grid will automatically generate rows of three columns depending on the number of items! We're also using grid-auto-rows to make each generated row have the height of 100 pixels.

4. The minmax() function

Most of the times we want our layout to adapt to the content it holds, but still need some control over it. The minmax() function is perfect for this, as it allows us to specify a minimum and maximum value for our rows or columns.

Fixed Rows

    
                 .grid-container {
                   display: grid;
                   grid-template-columns: repeat(3, 1fr);
                   grid-template-rows: 100px;
               gap:  1em;
                    
                }
Using Minmax()

    
                 .grid-container {
                   display: grid;
                   grid-template-columns: repeat(3, 1fr);
                   grid-template-rows: minmax(100px, auto);
               gap:  1em;
                    
                }

As you can see, the first row will get resized to fit the whole content, but the second row will maintain a min of 100px since no resizing is needed.

5. Superpower - Grid Template Areas

Template areas are a huge help when you want to create custom layouts using CSS Grid. We typically use this in combination with grid-template-columns and also grid-template-rows to achieve a better definition of space inside our grid layout!


    
                <div class="grid-container">
                  <div class="grid-item-1"> 1 </div>
                  <div class="grid-item-2"> 2 </div>
                  <div class="grid-item-3"> 3 </div>
                  <div class="grid-item-4"> 4 </div>
                </div>
              
              
            

    
               .grid-container {
                 display: grid;
                 grid-template-areas: 
            side header header
            side main main
            side main main
            footer footer footer
       grid-template-columns: 1fr 2fr 2fr;
                 grid-template-rows:1fr 1fr 2fr 1fr
       gap:  1em;
                  
              }
    .grid-item-1 {
       grid-area: side;
    }
    .grid-item-2 {
       grid-area: header;
            
              }
    .grid-item-3 {
       grid-area: main;
    }
    .grid-item-4 {
       grid-area: footer;
    }

Using template areas lets us describe exactly how we want our layout to look like! Using grid-area on the children elements, we can map them to the defined areas!

4. Flexbox Or Grid  - what to prefer?
  • Grid is for layout and Flexbox is for alignment.
  • Flexbox is designed for one dimension layout - either a row or a column.
  • Grid is designed for two dimension layout - rows and columns at the same time
  • Grid has space occupying features like fractional units, and the ability for content to break grids
  • Flex has unique features of flexbox that you can use for example: instead of using margin-right: auto use justify-content: flex-start or align-items: start.
